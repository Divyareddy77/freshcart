# 🛒 FreshCart

FreshCart is a responsive online grocery shopping web application built using **React**, **TypeScript**, and **Tailwind CSS**. It provides a smooth shopping experience with product browsing, cart management, user authentication, checkout, and order tracking.

---

## 🚀 Features

- 🥦 Browse Vegetables, Fruits, Dairy, Grocery & Non-Veg
- 🛒 Add/Remove items from Cart
- 🔐 User Registration & Login
- 📍 Auto Fetch Delivery Address using Location API
- 💳 UPI QR Code & Cash on Delivery
- 📧 Order Confirmation Email
- 📦 Order History
- 📱 Fully Responsive UI

---

## 🛠 Tech Stack

- React
- TypeScript
- Tailwind CSS
- React Router DOM

- React Hook Form
